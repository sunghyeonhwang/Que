Now

 Daily hub

오늘

내 일정, 내 작업, 체크인 응답, 주의 필요 항목을 하루 시작 화면으로 정리합니다.

 Calendar

캘린더

회사 일정, 작업, 마일스톤을 같은 시간축에 표시하고 작업 상세 패널을 함께 봅니다.

 All members

전체 캘린더

주간/월간 캘린더만 두고 8명 전체 멤버의 일정을 한 화면에서 확인합니다.

 Timeline

가로 캘린더

날짜를 가로축으로 두고 멤버와 캘린더 일정을 긴 막대 형태로 확인합니다.

 Attention

팀 현황

사람별 시간표와 문제발생, 홀드, 일정 충돌, 최근 변경 내역을 관리자가 빠르게 확인합니다.

 Heatmap

히트맵

멤버별 작업량이 많은 날짜와 여유 있는 사람을 색상과 막대 그래프로 확인합니다.

 Meeting notes

회의록

Plaud Note에서 내보낸 Markdown 회의록을 업로드하고 Action 추출 대상으로 관리합니다.

 Action

Action

회의록에서 추출된 Task 후보를 담당자와 마감일 기준으로 확인하고 캘린더에 반영합니다.

 Milestone

프로젝트

프로젝트 마일스톤, 연결 작업, 병목, 다음 액션을 하나의 운영 화면으로 묶습니다.

 Payment

결제

결제 요청을 등록하고 입금 후 완료, 대기, 취소 상태를 확인합니다.

캘린더는 운영 맥락

모든 데이터를 캘린더 이벤트로 저장하지 않고 작업, 일정, 마일스톤, 상태 로그를 분리하는 전제입니다.
