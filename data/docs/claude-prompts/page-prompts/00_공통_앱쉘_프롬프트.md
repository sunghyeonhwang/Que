# 공통 App Shell 개발 프롬프트

```text
Que 앱의 공통 App Shell을 구현해줘.

참고 문서:
- docs/que-product-plan.md
- preview/index.html
- preview/styles.css

목표:
- 모든 메뉴가 공유하는 레이아웃, 내비게이션, 상태 색상, 타이포그래피, mock data 기반을 만든다.
- 태블릿에서 메뉴 이동과 상태 확인이 편해야 하고, FHD에서는 정보가 넓게 보이도록 한다.

구현 범위:
- App Shell
- 반응형 사이드바 또는 접이식 내비게이션
- 페이지 헤더
- 공통 toolbar
- 상태 Badge
- Metric Card
- Task Row
- Page container
- ScrollArea 정책
- mock user/member/project/task/calendar data

메뉴:
- Now
- 오늘
- 캘린더
- 전체 캘린더
- 가로 캘린더
- 팀 현황
- 히트맵
- 회의록
- Action
- 프로젝트
- 결제

shadcn/ui:
- Button
- Badge
- Card
- ScrollArea
- Tooltip
- Separator
- Avatar
- Sheet 또는 Sidebar
- DropdownMenu

반응형 기준:
- 1920x1080: 사이드바 고정, 본문은 넓게 사용
- 1366x768: 사이드바 고정 가능하되 메뉴가 잘리면 내부 스크롤
- 1024x768: 사이드바 폭 축소 또는 collapsible
- 768x1024: 사이드바는 Sheet/Drawer 또는 상단 메뉴로 전환

상태 색상:
- green: 진행/완료
- blue: 예정/정보
- amber: 주의/대기
- red: 문제/취소
- violet: 회의록/응답대기

수용 조건:
- 모든 페이지가 동일한 App Shell 안에서 렌더링된다.
- 메뉴 active 상태가 정확하다.
- 태블릿 세로에서 메뉴 때문에 본문이 깨지지 않는다.
- 터치 대상은 최소 40px 이상이다.
- 공통 컴포넌트는 재사용 가능하게 분리되어 있다.
- lint/typecheck/build를 통과한다.
```
