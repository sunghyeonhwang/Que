# 팀 현황 페이지 개발 프롬프트

```text
Que의 팀 현황 페이지를 구현해줘.

참고:
- docs/que-product-plan.md의 팀 현황 정의
- preview/team.html

목적:
- 팀 전체의 오늘 업무 흐름과 병목을 확인한다.
- 감시 화면이 아니라 Attention Queue와 도움 요청을 중심으로 보여준다.

필수 UI:
- 상단 요약: 진행중, 문제발생, 홀드, 마감 임박, 최근 변경
- 사람별 시간표
- Attention Queue
- 일정 충돌 목록
- 최근 변경 내역
- 상태 상세 Sheet

shadcn/ui:
- Card
- Badge
- Table
- ScrollArea
- Sheet
- Button
- Avatar

반응형:
- FHD: 사람별 시간표와 Attention 패널 2열
- 태블릿 가로: 시간표는 넓게, Attention은 아래 또는 우측
- 태블릿 세로: 시간표 내부 가로 스크롤, Attention은 위쪽에 요약 카드로 노출
- 사람 이름/시간 헤더 sticky

상호작용:
- Attention 항목 클릭 시 상세 Sheet
- 문제발생/홀드 항목은 사유, 도움 필요한 사람, 다음 확인 시간을 표시
- 최근 변경은 관련 작업으로 이동할 수 있게 링크 구조를 둔다.

수용 조건:
- 어디가 막혔는지 첫 화면에서 보인다.
- 누가 일하고 있는지 감시하는 톤보다 병목 해결 톤이다.
- 태블릿에서 사람별 시간표가 깨지지 않는다.
- lint/typecheck/build 통과.
```
