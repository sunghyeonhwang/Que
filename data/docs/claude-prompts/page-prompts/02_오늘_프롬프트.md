# 오늘 페이지 개발 프롬프트

```text
Que의 오늘 페이지를 구현해줘.

참고:
- docs/que-product-plan.md의 오늘 화면 정의
- preview/today.html

목적:
- 로그인한 사용자가 하루에 한 번 열어 내 일정, 작업, 체크인 응답, 마감 임박, 문제 관련 항목을 확인한다.

필수 UI:
- 자연어 빠른 입력
- 자연어 해석 확인 카드
- 오늘 내 작업 타임라인
- 자동 체크인 응답 카드
- Attention/마감 임박 패널
- 최근 변경 또는 내가 관련된 문제 목록

자연어 예시:
- 내일 오후 3시에 황성현씨 상세페이지 QA 넣어줘

체크인 선택지:
- 작업중
- 완료
- 다른 작업 때문에 시간 변경 필요
- 문제발생
- 필요없어짐
- @다른프로젝트와 병합
- 나중에 답변

shadcn/ui:
- Input
- Button
- Card
- Badge
- Alert
- Dialog 또는 Sheet
- Select
- Sonner

반응형:
- FHD: 좌측 타임라인, 우측 체크인/주의 패널
- 태블릿 가로: 2열 가능
- 태블릿 세로: 타임라인 먼저, 체크인/주의 패널은 아래
- 자연어 입력창과 주요 버튼은 터치하기 쉽게 높이 44px 이상

상호작용:
- 입력 focus 또는 해석 버튼 클릭 시 확인 카드 표시
- 체크인 선택 버튼 클릭 시 상태 Badge 업데이트
- 문제발생/홀드 선택 시 추가 정보 Sheet를 열 준비 구조를 만든다.

수용 조건:
- 사용자가 오늘 할 행동을 첫 화면에서 알 수 있다.
- 상태 변경이 3초 안에 가능한 버튼 구조다.
- 태블릿 세로에서 입력창과 카드가 겹치지 않는다.
- lint/typecheck/build 통과.
```
