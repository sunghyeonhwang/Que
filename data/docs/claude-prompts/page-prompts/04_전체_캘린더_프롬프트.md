# 전체 캘린더 페이지 개발 프롬프트

```text
Que의 전체 캘린더 페이지를 구현해줘.

참고:
- docs/que-product-plan.md의 전체 캘린더 정의
- preview/all-calendar.html

목적:
- 전체 8명 멤버의 일정을 주간/월간 캘린더로만 확인한다.
- 상세 패널이나 운영 정보보다 일정표 자체에 집중한다.

필수 UI:
- 주간/월간 보기 토글
- 전체 멤버 범례
- 주간 보기: 사람별 행, 요일별 열
- 월간 보기: 날짜별 전체 멤버 일정 요약
- 회사 일정, Que 작업, 비공개 자리비움 표시
- 드래그 가능한 Que 작업/마일스톤

멤버:
- 황성현
- 오승훈
- 황성진
- 박승환
- 송수용
- 이예진
- 김리원
- 이혜진

shadcn/ui:
- Tabs
- Badge
- Button
- ScrollArea
- Tooltip

반응형:
- FHD: 8명 x 5일 이상을 넓게 표시
- 태블릿 가로: 사람 이름 첫 열 sticky, 날짜 헤더 sticky
- 태블릿 세로: 내부 가로 스크롤 필수
- 월간 보기 날짜 셀은 최소 폭을 유지

수용 조건:
- 전체 멤버의 일정 점유 상태가 빠르게 보인다.
- 개인 비공개 일정은 제목 없이 자리비움으로 표시된다.
- 캘린더 내부 스크롤이 있고 페이지 전체 레이아웃은 깨지지 않는다.
- lint/typecheck/build 통과.
```
