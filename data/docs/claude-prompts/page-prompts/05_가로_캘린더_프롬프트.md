# 캘린더 — 타임라인 뷰 개발 프롬프트

> 별도 메뉴/페이지가 아니라 **캘린더 페이지(03) 안의 "타임라인" 뷰**다. 03 프롬프트의 뷰 스위처 구조 위에 구현한다.

```text
Que 캘린더의 타임라인 뷰를 구현해줘.

참고:
- docs/que-product-plan.md의 "캘린더 — 타임라인 뷰" 정의
- preview/horizontal-calendar.html

목적:
- 날짜가 가로로 흐르는 타임라인에서 멤버별 일정과 프로젝트/캘린더 마일스톤을 확인한다.

필수 UI:
- 날짜 가로축
- 멤버별 일정 행
- 프로젝트/캘린더별 마일스톤 행
- 여러 날짜에 걸친 작업 막대
- 오늘 기준선
- 기간 필터: 7일, 14일, 30일
- 드래그 이동 가능한 작업/마일스톤

shadcn/ui:
- Button
- Badge
- ScrollArea
- Tooltip
- Sheet

반응형:
- FHD: 긴 타임라인을 넓게 보여준다.
- 태블릿 가로: 좌측 라벨 sticky, 타임라인은 가로 스크롤
- 태블릿 세로: 라벨 폭을 줄이고 막대 텍스트는 말줄임 처리
- 오늘 기준선은 스크롤 중에도 위치를 알 수 있어야 한다.

상호작용:
- 작업 막대 드래그로 시작 날짜 이동
- 마일스톤 드래그로 due date 변경
- 이동 후 변경 배지와 로그 반영

수용 조건:
- 일반 캘린더보다 기간 작업과 마일스톤 겹침을 보기 좋다.
- 긴 작업명은 레이아웃을 밀지 않고 말줄임 처리된다.
- 터치 환경에서 드래그가 어렵다면 대체 날짜 변경 Sheet를 제공할 구조를 둔다.
- lint/typecheck/build 통과.
```
