# Que 사용 예제 영상 — 제작 가이드 (도구·에이전트·촬영 시나리오)

> 목적: 프레젠테이션 슬라이드 9 "사용 동영상 (예제)" 제작.
> 결론 먼저: **① 지금 이 세션에서 바로 = Claude in Chrome `gif_creator`(GIF)** · **② 발표 품질 mp4 = Screen Studio(맥) 손촬영** · **③ 반복 자동화 = Playwright 녹화 스크립트**.

## 1. 도구·에이전트 옵션 비교

| 옵션 | 무엇 | 품질 | 비용/노력 | 추천 용도 |
|---|---|---|---|---|
| **A. Claude in Chrome — gif_creator** | 이 세션의 브라우저 에이전트가 Que를 직접 조작하며 GIF 녹화(`mcp__claude-in-chrome__gif_creator`). "체크인 흐름 GIF 만들어줘"라고 지시하면 됨 | GIF(발표 삽입 가능, 음성 없음) | **0 — 지금 바로** | 초안·사내 공유·PPT 삽입용 짧은 데모 |
| **B. Screen Studio** (screen.studio, 맥 유료 ~$89) | 손으로 조작하며 녹화 — 커서 확대·부드러운 줌이 자동. 발표용 데모 영상의 사실상 표준 | mp4 고품질 | 유료·10분 설치 | **발표 본편 영상(권장)** |
| C. QuickTime + iMovie (맥 기본) | 무료 화면 녹화+컷 편집 | mp4 보통 | 0 | 예산 0일 때 |
| D. OBS Studio (무료) | 방송급 녹화·장면 전환 | mp4 좋음 | 무료·설정 필요 | 길거나 멀티 장면 |
| E. Playwright 스크립트 녹화 | 코드로 조작을 재현하며 `recordVideo`로 webm 저장 — 레포에 스크립트를 두면 **버전 올라갈 때마다 데모 영상 자동 재생성** | webm→mp4 변환 | 개발 1~2h | 릴리즈마다 갱신되는 데모 |
| F. Remotion (React 코드 영상) | 코드로 모션그래픽 렌더 — 인트로/타이틀·번호 카운트업 | mp4 최고 | 개발 반나절+ | 인트로 5초·아웃트로(선택) |
| G. CapCut / DaVinci Resolve (무료) | 자막·배속·트랜지션 후반 편집 | — | 무료 | B~E 결과물 다듬기 |

**추천 조합**: A로 오늘 초안 GIF 3개 → 발표 확정되면 B(Screen Studio)로 본편 1~2분 재촬영 → G에서 자막만.

### 이 세션에서 쓸 수 있는 에이전트(요청만 하면 됨)
- **브라우저 에이전트 + gif_creator**: "Que에서 대화형 체크인 흐름을 GIF로 녹화해줘" → Chrome을 조작하며 `checkin_demo.gif` 생성. 프레임 여유를 넣어 부드럽게 뽑아준다.
- **Playwright 에이전트**: 데모 스크립트(로그인→조작→녹화) 작성 위임 가능 — 릴리즈 자동화 원하면 지시.

## 2. 촬영 시나리오 3편 (각 20~40초 — 콘티)

### 영상 A. "폰으로 30초 체크인" ⭐발표 오프닝 추천
1. (2s) Slack DM: "스탠드업 열렸습니다 [초안으로 시작]" 버튼
2. (3s) 폰 브라우저로 /daily 열림 — 하단 탭바 보임
3. (10s) [대화로 체크인] → AI: "어제 '배너 시안' 끝내셨네요…" → [그대로 쓸게요] → 막힘 [없어요] → 확인 카드 [제출]
4. (5s) 전원 카드에 내 체크인 반영 + "N/8 제출"
- 메시지 자막: "보고는 30초. 나머지는 AI가."

### 영상 B. "말로 작업 등록" (Copilot)
1. (3s) 아무 화면에서 ⌘K — 첫 항목 "✨ AI에게 묻기"
2. (5s) "오늘 팀에서 막힌 작업 뭐야?" → 실데이터 답변+출처 칩
3. (10s) "내일 오전 10시에 광고 리포트 정리 작업 잡아줘" → **확인 카드**(제목·담당·기한) → [실행]
4. (5s) /today에 작업 생김 + 기록(via: chat)
- 자막: "실행은 언제나 사람이 확인한 후에."

### 영상 C. "간트에서 회의 끝내기"
1. (4s) gant.griff.co.kr(회의실 화면) — 전 프로젝트 통합 간트, 위험만 보기 ON
2. (8s) 주의 마일스톤 칩 클릭 → 맥락 확인 → **드래그로 기한 이동**(스냅)
3. (5s) 하단 "오늘 조정 2건" → [회의록 초안으로] → 회의록 생성 토스트
- 자막: "결정이 곧 기록이다."
